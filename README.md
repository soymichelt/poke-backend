# Poke Backend

## Descripción

Backend desarrollado con [Nest](https://github.com/nestjs/nest), el cuál implementa la API de Pokemon para servir información.

## Inicialización del proyecto

```bash
$ npm install
```

## Compilación y ejecución del proyecto

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Ejecución de tests

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
